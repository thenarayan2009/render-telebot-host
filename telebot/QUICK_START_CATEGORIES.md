# Quick Start Guide - Task Categories

## For Users

### How to Browse Tasks by Category

1. **Start the bot**: Press `/start` or click "🎯 All Tasks" button
2. **Select a category**:
   - 📥 Download & Register - Download apps and register accounts
   - 📢 Join Telegram Channel - Join required channels
   - 👥 Follow Social Media - Follow social media accounts
   - 🔗 Referral Tasks - Refer friends and earn
   - 🎯 General - Other tasks

3. **View tasks**: See all available tasks in the selected category
4. **Select a task**: Click on a task to get details and complete it
5. **Go back**: Use "🔙 Back to Categories" to browse other categories

---

## For Admins

### How to Add a Task with Category

1. **Go to admin panel**: `/admin`
2. **Click "🎯 Manage Tasks"**
3. **Click "➕ Add New Task"**
4. **Follow the prompts**:
   - Enter task title
   - Enter task description
   - Enter task link
   - Enter reward amount (in rupees)
   - Enter quantity (how many users can complete)
   - **Select or enter category** (new step!)
     - Choose from: Download & Register, Join Telegram Channel, Follow Social Media, Referral Tasks, General
     - Or type a custom category name

5. **Task is created** and users are automatically notified!

### How to Manage Categories

1. **Go to admin panel**: `/admin`
2. **Click "🎯 Manage Tasks"**
3. **Click "📂 Manage Categories"**
4. **View all categories** with task counts
5. **Edit categories** (click on a category to edit)

## Category Descriptions

### 📥 Download & Register
Tasks where users need to:
- Download an app
- Register an account
- Complete signup process
- Example reward: ₹5-10

### 📢 Join Telegram Channel
Tasks where users need to:
- Join a Telegram channel
- Subscribe to updates
- Follow channel rules
- Example reward: ₹2-5

### 👥 Follow Social Media
Tasks where users need to:
- Follow on Instagram, Twitter, Facebook, etc.
- Like/Share posts
- Subscribe to accounts
- Example reward: ₹3-8

### 🔗 Referral Tasks
Tasks where users need to:
- Refer friends to the bot
- Get referral bonuses
- Earn from friend's activity
- Example reward: ₹2-5 per referral

### 🎯 General
Any other tasks that don't fit above categories

## Troubleshooting

**Q: I don't see categories**
A: Make sure you have active tasks added. Categories only appear if tasks exist.

**Q: How do I change a task's category?**
A: Edit the task through admin panel and update the category.

**Q: Can I create custom categories?**
A: Yes! Just type any category name when adding a task.

**Q: Why is a task not showing in my category?**
A: The task might be inactive, completed, or quantity limit reached.
